# Expense Tracker (Flask)

A simple web app to track expenses with full CRUD operations.

## Features
- Add expense
- Edit expense
- Delete expense
- View all expenses

## Tech Stack
- Python (Flask)
- HTML, CSS
- JSON (data storage)

## Notes
- Backend logic built from scratch
- CSS/UI styled using AI assistance (Antigravity)

## Run locally
```bash
pip install flask
python app.py
